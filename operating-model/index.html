Size of the prize & feasibility — the prioritization criteria

Every opportunity surfaced in the process review gets scored on these two axes before it becomes a candidate initiative.

This matrix identifies candidates — it doesn't fund them. Each one still needs a real business case: an MVP-level financial model (cost, ROI, payback), built with Finance, before it gets resourced.

Size of the prize

  • Estimated $ impact — cost saved plus revenue potential
  • Volume of content or teams it touches
  • Leverage — does it benefit one team, or many via shared infrastructure like the creative studio

Feasibility

  • Technical complexity and data readiness — rules-based vs. requires judgment
  • Brand-risk exposure if something goes wrong
  • Change required — new roles, retraining, process redesign needed
  • Can it show a validated result inside the 90-day window — if not, it drops into Big Bets regardless of prize size
